# passwordgen
Password Generator é uma pequena GUI que gera senhas aleatórias de tamanho de 1 a 20 com possibilidade de customização (maíusculas, minúsculas, números e caracteres especiais)

# Bibliotecas utilizadas
* PyQt5 
* Pyperclip
* Random
